---
title: Project Configuration
---

Before starting the application configure few variables in the .env file

```shell
#ENVIRONMENTS
PORT=

#DATABASE ENVs
DATABASE_URL=

#JWT
JWT_SECRET=
JWT_EXPIRES_IN_DAYS=

WEBHOOK_SECRET=

#password length
MIN_PASSWORD_LENGTH=
MAX_PASSWORD_LENGTH=

RESET_PASSWORD_SLUG=
INVITE_SLUG=

ENVIRONMENT=
```
